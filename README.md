Clone o repositório:
```bash
git clone https://github.com/brunomat480/inciBrasil-challenge.git
```

Instale as dependências:
```bash
npm i
```

Inicie a aplicação:
```bash
npm run dev
```
##

Execute as migrações:
```bash
npm run db:migrate
```

##

Execute o seed:
```bash
npm run db:seed
```

##

#### Adicionais
